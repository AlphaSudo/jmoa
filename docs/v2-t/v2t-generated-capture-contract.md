# V2-T Generated Capture Contract

A valid future V2-T capture bundle contains the exact static artifact SHA-256,
the same SHA-256 recorded by each capture manifest, and separate startup,
warmup, and workload class-load/histogram attribution reports. Diagnostic
logging remains outside V2-C memory confirmation pairs.

The report is matched only when all four hashes are identical and all lifecycle
captures are present. A capture may still be useful for analysis when this
fails, but it is labeled incomplete and cannot alter prototype admission.
