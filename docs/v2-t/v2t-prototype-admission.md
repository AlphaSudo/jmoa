# V2-T Prototype Admission

V2-T admits no generated-family prototype.

The gate is conjunctive: exact static/capture artifact fingerprint equality,
three lifecycle diagnostic captures, runtime relevance, an allowed safety
category, a family-specific semantic plan, and a measurable ROI must all be
present. V2-T currently has partial evidence only, so every family remains
report-only or blocked according to the V2-S safety model.

This is a valid outcome. It prevents a V2-S fixture or an unmatched historical
capture from becoming a source-level mutation decision.
