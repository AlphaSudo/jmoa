# Doctor CDS Positive Control

Doctor remains the positive control for a successful CDS policy: three paired wins, median PSS `-5,156 KB`, Private_Dirty `-5,212 KB`, and memory.current `-6,975,488 bytes`.

The transferable method is strict variant-specific archive training, artifact/archive pairing, runtime materialization proof, and non-perturbing memory pairs. The performance result itself is not transferable: Doctor proves CDS can be right for one service, not that it is right for Patient.
