# v0.8.3 - V2-K Doctor Unblock Gate

This release tags the policy-aware Doctor runtime unblock gate.

It adds:

```text
scripts/doctor-runtime-unblock-gate.ps1
```

The gate verifies the known Doctor D2 artifacts and blocks before smoke when
the runtime stack is incomplete.

Current blocker classes:

```text
D2R_CDS_NOT_TRAINED
MISSING_IMAGE
MISSING_CONFIG
MISSING_DATABASE
MISSING_NETWORK
PORT_BUSY_OR_BLOCKED
BLOCKED_PRIVATE_STACK
```

This release does not contain a Doctor semantic smoke, runtime screen, V2-C
confirmation, V2-D attribution, startup claim, CDS claim, or runtime memory
claim.
