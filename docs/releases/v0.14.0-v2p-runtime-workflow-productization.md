# v0.14.0 V2-P Runtime Workflow Productization

V2-P adds a script-first, evidence-first runtime workflow coordinator.

It normalizes recommendation, preflight, materialization, semantic smoke,
screen, confirmation, V2-C, V2-D, and claim-register gates into one report.
It also adds a historical state-machine replay and a claim-register consistency
guard. This release creates no new runtime performance claim.
