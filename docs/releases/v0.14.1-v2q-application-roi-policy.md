# v0.14.1-v2q-application-roi-policy

This release closes V2-Q as an application-class admission and ROI-policy
guardrail.

V2-Q result:

```text
artifact and semantic gates: passed
diagnostic rerun: passed
3-pair confirmation: failed
paired wins: 1/3
median PSS delta: +5,732 KB
median Private_Dirty delta: +5,760 KB
median memory.current delta: +5,922,816 bytes
runtime claim: false
```

The important product rule is:

```text
tiny application-class raw reduction must not inherit dependency-reducer confidence
```

Application-class raw reduction remains artifact/semantic-only unless removable
application metadata reaches at least `32 KB`, at least `50` application classes
are reduced, or the exact application scope earns runtime evidence.

This release does not claim generated-class mutation, application-class runtime
improvement, startup improvement, or cross-service generalization.
