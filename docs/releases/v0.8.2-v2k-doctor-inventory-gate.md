# v0.8.2 - V2-K Doctor Inventory Gate

This release tags the first executable V2-K Doctor runtime unblock gate.

It adds:

```text
scripts/doctor-runtime-inventory.ps1
```

The gate converts Doctor from a vague runtime blocker into concrete missing
inputs:

```text
MISSING_IMAGE
MISSING_CONFIG
MISSING_DATABASE
D2R_CDS_NOT_TRAINED
BLOCKED_PRIVATE_STACK
```

This tag does not contain a Doctor semantic smoke, runtime screen, V2-C
confirmation, V2-D attribution, startup claim, CDS claim, or runtime memory
claim.
