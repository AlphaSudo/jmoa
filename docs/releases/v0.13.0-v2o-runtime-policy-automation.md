# v0.13.0-v2o-runtime-policy-automation

JMOA V2-O adds report-only automation around the existing runtime-policy and
evidence engines.

- Adds `jmoa:runtime-preflight` with SHA-backed artifact/CDS checks.
- Adds explicit CDS training, materialization proof, semantic smoke, paired
  screen capture, and V2-C/V2-D orchestration helpers.
- Standardizes `bN`/`cN` evidence folders for V2-C.
- Preserves exact CDS artifact/archive pairing rules and blocks policy mixing.

No reducer type, runtime configuration, or runtime performance claim is added.
