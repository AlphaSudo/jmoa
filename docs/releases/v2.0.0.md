# JMOA v2.0.0

Stable `v2.0.0` closes the V2 launch matrix under service-specific confirmed
runtime policies.

| Service | Confirmed policy | Median V1-to-V2 PSS |
| --- | --- | ---: |
| PetClinic customers-service | `NO_CDS_LOW_DIRTY` | -6,012 KB |
| Doctor-service | `CDS` | -5,156 KB |
| Patient-service | `NO_CDS_LOW_DIRTY` | -8,903 KB |

The release does not claim one universal runtime policy. Patient's CDS path
remains blocked, and the post-release dynamic AppCDS economics study does not
change the confirmed Patient no-CDS result.

Release: https://github.com/AlphaSudo/jmoa/releases/tag/v2.0.0
