# v0.11.0 - V2-M Reducer Recommendation Engine

V2-M adds a report-only admission and recommendation layer for the productized
raw LVT/LVTT reducer.

New Maven goal:

```text
jmoa:recommend-reducer
```

Capabilities:

```text
normalize reducer, audit, safety, semantic, V2-C, and V2-D reports
classify confirmed, screen-required, artifact-only, blocked, and diagnostic states
separate recommendation decision from public/private confirmation scope
require exact service, launch-mode, and runtime-policy scope for confirmed use
replay historical decisions with mismatch failure
```

Historical replay:

```text
V2-I customers raw: RECOMMEND_CONFIRMED / PUBLIC
V2-K Doctor raw: RECOMMEND_CONFIRMED / PRIVATE
V2-L visits raw: RECOMMEND_CONFIRMED / PUBLIC
V2-H hardened ASM: BLOCK_RUNTIME_PROMOTION
V2-J Doctor pre-runtime: ALLOW_ARTIFACT_ONLY
passed: 5/5
```

The release does not add a reducer, broaden metadata stripping, mutate bytecode,
or make a new runtime performance claim.
