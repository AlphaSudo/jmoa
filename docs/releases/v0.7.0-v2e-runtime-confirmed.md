This release confirms V2-E as the first controlled post-v1 runtime reducer for JMOA.

```text
service: Spring PetClinic customers-service
comparison: full P2 vs full P2 + V2-E reducer

MALLOC_ARENA_MAX: 1

```

## Result

```text
valid runs: 6/6

median memory.current delta: -12,255,232 bytes
workload errors: 0
```

## What Changed

V2-E adds an opt-in `release-low-footprint` reducer that strips only:

```text
LocalVariableTable
LocalVariableTypeTable
```

The reducer preserves:

```text
LineNumberTable
SourceFile
StackMapTable
RuntimeVisibleAnnotations
RuntimeInvisibleAnnotations
Signature
BootstrapMethods
module-info.class
```

## Claim Boundary

This release does not claim startup improvement, fat-JAR improvement, CDS/AppCDS improvement, cross-service generalization, or broad debug metadata stripping safety.
