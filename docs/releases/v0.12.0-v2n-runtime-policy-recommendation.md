# v0.12.0 - V2-N Runtime Policy Recommendation

V2-N adds a report-only runtime-policy recommendation engine for the existing
raw reducer.

New Maven goal:

```text
jmoa:recommend-runtime
```

Capabilities:

```text
exact protocol registry matching
public/private policy scope
CDS artifact/archive mismatch detection
no-CDS versus CDS comparison protection
analyze and preflight reports
historical replay with mismatch failure
```

Historical runtime replay passes 7/7 cases, including the Doctor D2R fresh-CDS
policy, the stale D2 archive block, mixed-policy block, and V2-H failed-screen
block.

This release does not add a reducer, mutate bytecode, train or reuse CDS
archives, change deployment configuration, or make a new runtime performance
claim.
