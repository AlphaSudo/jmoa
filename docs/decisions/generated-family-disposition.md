# Generated-Family Disposition

Decision: no new generated/proxy family mutation is admitted in V2.

Static inventory, runtime relevance, and matched lifecycle evidence were
completed across three services. Lambda sites belong to the existing optimizer
domain. Spring Data and synthetic/bridge surfaces did not expose a bounded safe
transform. Spring AOT, CGLIB, JDK proxy, ByteBuddy, and Hibernate proxy families
remain identity, reflection, interception, or classloader sensitive.

The inventory and safety taxonomy remain shipped report-only capabilities. A
high class count or runtime relevance score is not sufficient admission evidence.
