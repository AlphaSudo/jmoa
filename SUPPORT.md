# Support

JMOA V2 is an Apache-2.0 research/tooling release. Use GitHub Issues for
reproducible build defects, unsafe reducer behavior, or documentation gaps.

Include the JMOA version, JDK/Maven versions, launch mode, runtime policy,
artifact hashes, and the smallest publish-safe report that reproduces the
problem. Do not attach credentials, private service source, database dumps,
heap dumps, JFR files, or raw internal evidence.

Performance questions require V2-C-valid paired evidence. A single screen is
useful diagnostics but is not accepted as a performance claim.
